A deeply immersive, cinematic 3D developer portfolio showcasing full-stack engineering capabilities, real-world AI integrations, and professional achievements. 

## 🌌 Core Features

- **Cinematic Dark-Mode Aesthetics**: Unbounded typography, glassmorphism elements, and smooth gradient glows.
- **Interactive 3D Engine**: Utilizes Three.js / React Three Fiber for dynamic background rendering and floating geometry.
- **Dynamic Projects Showcase**: Detailed metrics, latency stats, and AI models for robust projects like *Dr. Brinjal* and *Sanchay AI*.
- **Integrated Live Contact Form**: Seamless, secure email routing powered by **EmailJS** without requiring a dedicated backend endpoint.
- **Automated Routing**: Smooth-scrolling architectural layout linking directly to sections (Hero, About, Work, Skills, Experience, Writing, Contact).
- **Responsive "Connect" Slider**: Cinematic, card-less UI for teammate testimonials with integrated LinkedIn navigation.

## 🚀 Local Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/SoulSaviour1234/3D-Portfolio.git
   cd 3D-Portfolio
   ```

2. **Navigate to the Frontend Workspace**
   ```bash
   cd frontend
   ```

3. **Install Dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Run the Development Server**
   ```bash
   npm start
   ```
   The platform will instantly hot-reload at `http://localhost:3000` (or `3001` depending on port availability).

## 🗃️ Data Structure (`mock.js`)

The entire application is driven by a central, single-source-of-truth configuration file located at `frontend/src/mock.js`. 

To update the portfolio in the future, you do not need to hunt down individual React components. Simply open `mock.js` and modify the JSON objects for:
- Profile details & Resumé PDF mappings
- Skill proficiencies
- Project data & metrics
- Work Experience & Education
- Blog articles
- Teammate testimonials
